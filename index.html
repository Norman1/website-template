About

This is an HTML template for text heavy documentation projects suiting my needs. I want a minimalistic and professional looking design while not using any CSS library. The design is in the form of the holy grail layout.

The following sections are both for documentation and for me to test the design against what is specified here.

Header

The header is at the top of the page and is not sticky.

Left Navbar

The left navbar is there to add links to other pages on the same website. The currently active page needs to get manually highlighted. When the user scrolls down, the content of the left navbar sticks to the top.

Right Navbar

The content of the right navbar should match the section headings. The user can use this navbar to jump within the page.

Article Area

Use HTML section elements to break down the content.

Footer

The footer is at the bottom of the page and is not sticky. If the content area is not big enough to fill the whole page, the footer gets pushed to the bottom of the viewport.

Responsiveness

For smaller devices, the left navbar gets put on the top of the article area while the right navbar disappears.
